Publications

Deraining of Image using UNet based Conditional Generative Adversarial Network

Samprit Bose, Deep R. Chavan, Maheshkumar H. Kolekar
[EAI International Conference on Cognitive Computing and Cyber Physical Systems, Accepted]

Proposed an approach for single image deraining based on conditional Genrative Adverserial Network. Tested on various synthetic and real-world datasets and achieved better results than state-of-the-art methods.

Conversion of Hand Gesture to Speech for Speech Im-paired People.

Deep R. Chavan, Maheshkumar H. Kolekar
[INDICON 2022 IEEE 19th India Council International Conference, Accepted]

Developed a novel architecture which helps for conversion of hand gesture to text. The proposed method is capable to convert entire sentence to speech using hand gesture.

Vehicle Classification and Counting for Traffic Video Monitoring Using YOLO-v3.

Samprit Bose, Chavan Deep Ramesh, Maheshkumar H. Kolekar
[International conference on connected systems intelligence (CSI’22), Accepted]

Developed a novel architecture for vehicle counting. The proposed method achieved 94% accuracy in real video dataset and average 97.4% accuracy on M-30 and M-30 HD publically available datasets.

Projects

Autonomous Vehicle Model

Led a team of seven members to develop an end-to-end model for an Autonomous Vehicle, utilizing a single RGB camera for perception and generating steering angle and throttle inputs from captured images.

Object Tracking and Depth Estimation Using Sensor Fusion

Utilized YOLO-NAS for object detection, DeepSORT for object tracking, and sensor fusion techniques to combine RGB camera data with depth information for accurate object identification and depth estimation in the Autonomous Vehicle model.

3D Object Detection

Implemented real-time 3D object detection using the RTM3D paper, a monocular camera based 3D object detector. Used carla-ros-bridge to extract data from Carla simulator and publish the detected images.
